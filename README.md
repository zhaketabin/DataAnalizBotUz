# 📊 DataAnalizBotUz

Telegram бот для анализа данных компаний.

## Настройка

### 1. Переменные окружения (Railway → Variables)
```
BOT_TOKEN=ваш_токен_от_BotFather
GEMINI_API_KEY=ваш_ключ_от_aistudio.google.com
ADMIN_ID=ваш_telegram_id
```

### Как узнать свой ADMIN_ID?
Напишите боту @userinfobot в Telegram — он покажет ваш ID.

---

## Команды администратора

| Команда | Описание |
|--------|----------|
| `/activate ZH-705000001` | Активировать клиента на 1 месяц |
| `/activate ZH-705000001 3` | Активировать на 3 месяца |
| `/deactivate ZH-705000001` | Отключить клиента |
| `/clients` | Список всех клиентов |
| `/adminhelp` | Помощь |

---

## Как работает бот

1. Клиент пишет /start
2. Выбирает язык (Ўзбекча / Қазақша / Русский / English)
3. Получает уникальный код ZH-705XXXXXX
4. Если не оплатил — бот не работает
5. После оплаты: /activate ZH-705XXXXXX → бот работает
6. Клиент выбирает вопрос → отправляет Excel/CSV → получает анализ
